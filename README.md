# How to USE

-------------------
chnroute mode:
'sh /etc_ro/v2ray/whitelist.sh'

---------------------
gfwlist mode:
'sh /etc_ro/v2ray/gfwlist.sh'

Replace your information >> config.json & gfwconfig.json

If v2ray is replaced, remember to change "/etc_ro/v2ray/" in whitelist.sh & gfwlist.sh
